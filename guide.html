사용자 가이드

1. 계정 생성

이메일 주소와 암호를 해당 칸에 입력하고 하단의 '이메일 회원 가입' 버튼을 클릭하거나 '구글 로그인' 버튼을 클릭하여 계정을 생성합니다.

2. 로그인

1단계에서 이메일 주소와 암호로 계정을 사용하여 계정을 생성한 경우, 해당 이메일 주소와 암호를 입력하고 하단의 '이메일 로그인' 버튼을 클릭하여 로그인합니다. '구글 로그인' 방식으로 계정이 만들어진 경우는 '구글 로그인' 버튼을 클릭하여 로그인을 진행합니다. 이미 로그인이 된 상태이면 자동으로 로그인 처리됩니다.

3. 기본 정보 설정

2단계의 로그인이 완료되면 사용자의 기본 정보를 설정하는 화면에서 간략하게 사용자에 대한 정보를 입력하고 상단의 저장 아이콘 버튼을 클릭하여 저장합니다.
