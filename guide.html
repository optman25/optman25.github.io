사용자 가이드

1. 계정 생성

이메일 주소와 암호를 해당 칸에 입력하고 하단의 '이메일 회원 가입' 버튼을 클릭하거나 '구글 로그인' 버튼을 클릭하여 계정을 생성합니다. (계정을 최초 생성하면 '게스트' 계정으로 생성되고 앱의 기능을 정상적으로 사용하기 위해 일반 계정으로 전환이 필요하니 개발자(optman25@gmail.com)에게 이메일로 연락하여 계정 전환을 진행해야 합니다.)

2. 로그인

1단계에서 이메일 주소와 암호로 계정을 사용하여 계정을 생성한 경우, 해당 이메일 주소와 암호를 입력하고 하단의 '이메일 로그인' 버튼을 클릭하여 로그인합니다. '구글 로그인' 방식으로 계정이 만들어진 경우는 '구글 로그인' 버튼을 클릭하여 로그인을 진행합니다. 이미 로그인이 된 상태이면 자동으로 로그인 처리됩니다.

3. 기본 정보 설정

2단계의 로그인이 완료되면 사용자의 기본 정보를 설정하는 화면에서 간략하게 사용자에 대한 정보를 입력하고 상단의 저장 아이콘 버튼을 클릭하여 저장합니다. 사용자 이름과 건물 이름, 건물 주소, 호실 갯수를 입력하여 기본 정보를 설정합니다.

4. 건물 상세 정보 설정

3단계에서 사용자에 대한 기본 정보의 설정이 완료되면 건물의 상세 정보를 입력할 수 있습니다. 1. 건물 유형을 선택합니다. 현재는 하나의 건물 유형인 '원룸'을 선택할 수 있습니다. 향후, 관리자에 의해 다양한 건물의 유형이 추가될 예정입니다. 2. 선택한 건물 유형이 갖는 호실 유형을 선택합니다. 1항에서 건물 유형으로 원룸을 선택했다면 호실의 유형 중에 원룸, 투룸, 미니 투룸을 선택합니다. 다수 선택 가능합니다. 3. 기본 정보에서 설정한 호실의 갯수만큼 호실의 이름과 호실의 유형을 선택할 수 있는 화면이 나타납니다. 호실의 이름을 입력하고 호실의 유협을 드랍다운 메뉴를 사용하여 선택합니다. 4. 각 호실 유형에 대하여 호실에 포함된 옵션의 이름과 종류를 추가하거나 삭제할 수 있습니다. 5. 하단의 화면에는 각 옵션에 대해 점검할 항목과 각 항목 당 선택하여 지정할 수 있는 점검 값의 목록을 추가하거난 삭제할 수 있습니다. 설정이 완료되면 상단의 저장 버튼을 클릭하여 저장합니다. 처음부터 다시 진행하려면 '초기화' 버튼을 클릭하여 정보의 설정을 초기화합니다.

5. 점검 내역 작성

3, 4단계를 거쳐 건물에 대한 정보를 설정하면 각 호실의 옵션에 대한 점검 내역을 작성할 수 있습니다. 주 화면(아래에 '목록' 버튼과 '달력' 버튼이 있어 점검 목록에 표시를 전환할 수 있는 화면이 주화면임) 하단의 중앙에 있는 '+' 버튼을 클릭하여 점검내역을 작성합니다. 호실과 옵션을 드랍다운 목록에서 선택하면 옵션에 대한 점검항목이 표시되고 점검항목별로 선택가능한 상태값 목록에서 해당되는 상태값을 선택하여 옵션의 현재 상태를 설정합니다. 추가적이 설명이 필요한 경우, 하단의 빈 칸에 내용을 간단히 작성할 수 있으며, 상단의 '카메라' 버튼을 클릭하여 카메라로 촬영하거나 갤러리에 포함된 사진을 관련 이미지로 추가할 수 있습니다. 상단의 '저장' 버튼을 클릭하면 점검내역의 작성이 완료되고 방금 작성한 점검내역이 포함된 목록이 주 화면에 표시됩니다.
